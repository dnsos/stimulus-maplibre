# Stimulus MapLibre controller

A Stimulus controller that registers a MapLibre map with some configurations.

## Developing

Install dependencies:

```bash
yarn install
```

Start a development server:

```bash
npm run dev
```

Run the tests:

```bash
npm run test
```
